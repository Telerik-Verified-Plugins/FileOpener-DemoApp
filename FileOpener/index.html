File Opener Plugin Demo

The File Opener plugin allows you to open a local or remote file and ask the device which app(s) can handle the filetype.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS / Android - App package.

Fetching and opening files

On Android loading the local file from the www folder may fail because of a bug in older Cordova versions. Loading local files from the SD card will work though, as well as remote files - so it's not a plugin issue in case you wondered.
